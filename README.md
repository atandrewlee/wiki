# Dreamcatcher

This is my personal wiki website (documentation, information, etc.).

Tech Stack + Tools
- Hugo
- Obsidian
- Nginx


## Content
Content for this website (wiki) is located in another repository (dreamcatcher) as a submodule.

To update content: `git submodule update`

## Commands
- Create new content: `hugo new <file>`
- Preview site locally: `hugo server --buildDrafts --disableFastRender`
- To build the website: `hugo`
